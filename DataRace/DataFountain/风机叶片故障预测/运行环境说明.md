## 运行环境说明

1. windows 10
2. anaconda 4.5.4
3. python 3.6.5

### 项目中使用到的工具包

1. zipfile 读取zip文件
2. pandas 数据转换，操作
3. StratifiedKFold  交叉验证数据拆分
4. f1_score  计算F1-Score
5. lightgbm 机器学习库
6. roc_auc_score  计算AUC数值
7. numpy 计算库

如果没有对应的工具包，请先安装对应的安装包。
> python -m pip install zipfile

> python -m pip install lightgbm

> python -m pip install pandas
